# xhttp

HTTP/1.1 server implementation from scratch in Rust. No external dependencies, only the standard library.

WORK IN PROGRESS :)

## Project Scope

- [x] HTTP Streams / TCP
- [x] Request parsing (line, headers)
- [x] HTTP Responses
- [x] HTTP Body (request body with Content-Length)
- [x] Router with closures
- [x] Static files + MIME types
- [x] Chunked Encoding
- [x] Binary Data
- [ ] Thread pool (concurrency)
- [ ] Better error handling
